My Family Tree

Welcome to my family tree project. This is the front-end that will eventually connect to my SQL database and backend code.

About This Project

This site is built using simple HTML and CSS. The goal is to show a clean family tree layout with basic information.

Later, the data will come from SQL tables stored in the backend.

Current Features

  • Simple clean white layout
  • Basic information sections
  • Easy to expand into a real family tree viewer
